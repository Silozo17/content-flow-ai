# Netlify Backend Pack

This adds three simple backend endpoints to your Netlify site:

- `/health`
- `/api/test`
- `/api/test/connection`

## How to use (no coding)

1) Download the ZIP.
2) Unzip it.
3) In your GitHub repo, click **Add file â†’ Upload files**.
4) Drag the **netlify** folder and the **netlify.toml** file into GitHub (drop them at the repo ROOT).
5) Commit the changes.
6) In Netlify, trigger a new deploy. Done.
